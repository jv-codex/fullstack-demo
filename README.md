# Fullstack Demo

A full‑stack demo application built with **Angular 21**, **Spring Boot**, and **PostgreSQL**.  
This project demonstrates a modern end‑to‑end architecture with JWT authentication, REST APIs, and CRUD operations. Inspired by concepts from a *28-Minute* course and fully updated with my own implementation and tooling.

---

## Tech Stack

- **Frontend:** Angular 21  
- **Backend:** Spring Boot  
- **Database:** PostgreSQL  
- **Languages:** TypeScript & Java  
- **Tools:** STS, Node.js, npm  

---

## Features

- User authentication (JWT)  
- Task management (CRUD operations)  
- RESTful API design  
- Modular frontend and backend structure  
- PostgreSQL persistence layer
